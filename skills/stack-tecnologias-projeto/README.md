# Skill: Stack Tecnologias Projeto

Esta skill fornece orientações sobre as tecnologias e padrões de desenvolvimento utilizados no projeto Laravel com TALL Stack.

## Descrição

Esta skill ajuda a entender e seguir os padrões arquiteturais do projeto, que utiliza:

- TALL Stack (Tailwind CSS, Alpine.js, Livewire, Laravel)
- Componentes Livewire em vez de controllers tradicionais
- Integração com Alpine.js e SweetAlert para experiências de usuário dinâmicas

## Quando usar

Use esta skill quando:

- Estiver desenvolvendo novos recursos no projeto
- Precisar entender a arquitetura do sistema
- Consultar melhores práticas de desenvolvimento
- Ensinar novos membros da equipe sobre as tecnologias utilizadas
- Consultar como as rotas estão estruturadas com o padrão CRUD completo em componentes Livewire
